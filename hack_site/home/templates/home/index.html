About

This Django Website Application was made as a future solution to be used in conjunction with future Bowling Green, KY transportation upgrades to allow for residents an easier way to access newly built subway/railway stations. These stations would not only allow residents another option for city travel, but would also attract new businesses to build alongside these station hubs for increased city growth.

Software Tools

Our team utilized several different software tools, packages, and APIs to accomplish this site. The following is a list:

  • Python 3.12.3
  • Django==5.0.6
  • django-easy-maps==1.1.3
  • djangorestframework==3.15.1
  • Bootstrap for styling
